# 隐私政策 — Platrare

**生效日期：** 2026年4月12日

Platrare 是一款以本地存储为核心的个人财务应用程序。本政策说明应用访问的数据、数据的使用方式以及您的权利。

---

## 1. 关于我们

Platrare 由个人开发者发布。联系方式可在 App Store 或 Google Play 商店页面找到，也可通过应用内的**设置 → 关于 → 复制支持信息**获取。

---

## 2. 存储在您设备上的数据

您在 Platrare 中创建的所有数据**仅存储在您的设备上**。我们不运营接收或存储您财务信息的服务器。

**本地存储的内容：**

| 类别 | 详情 |
|---|---|
| 财务账本 | 账户、实际余额、透支限额、交易历史、计划交易和类别 |
| 附件 | 您选择添加到交易中的收据和文件照片 |
| 偏好设置 | 基础货币、次要货币、主题、语言、余额可见性设置 |
| 安全 | 应用锁定状态；PIN 的单向加密哈希（原始 PIN 从不存储） |
| 汇率缓存 | 从第三方 API 下载并在本地缓存的公开货币汇率数据 |

---

## 3. 通过互联网发送的数据

### 3.1 汇率

为以多种货币显示金额，应用会定期从 **Frankfurter API**（api.frankfurter.dev / api.frankfurter.app）获取公开汇率数据，该 API 发布来自**欧洲中央银行 (ECB)** 的数据。这些请求：

- **不包含任何个人信息** — 仅为标准匿名 HTTP 请求。
- **不包含**您的账户名称、余额、交易或任何其他账本内容。
- 为减少网络活动，在设备上缓存最多 **6 小时**。

Frankfurter 服务器可能记录的任何数据（例如 HTTP 访问日志中的 IP 地址）受 Frankfurter 自身隐私政策的约束。

### 3.2 无分析、无广告

Platrare **不包含任何分析 SDK、崩溃报告服务或广告网络**。我们不收集或传输使用数据、设备标识符或行为遥测数据。

---

## 4. 设备权限

| 权限 | 用途 | 请求时机 |
|---|---|---|
| 相机 | 拍摄收据照片 | 仅在附件面板中点击"拍照"时 |
| 照片库 | 选择图像或文件进行附加 | 仅在点击"从相册选择"时 |
| 文件 | 附加 PDF 和其他文档 | 仅在点击"浏览文件"时 |
| 生物识别 / Face ID | 启用应用锁定时解锁应用 | 仅在显示锁定屏幕时 |
| 网络 | 获取货币汇率 | 自动在后台运行；不发送个人数据 |

应用**不会**请求访问您的位置、联系人、麦克风、日历或上述未列出的任何权限。

---

## 5. 应用锁定和生物识别认证

在设置中启用**打开时锁定应用**后：

- 应用使用操作系统安全生物识别框架（iOS LocalAuthentication / Android BiometricPrompt）。您的指纹或面部数据完全在 **OS 安全隔区**内处理 — 应用从不查看、存储或传输生物识别数据。
- 如果您设置了 PIN，只有该 PIN 的**单向加密哈希**会存储在应用专用设备存储中。原始 PIN 从不写入磁盘。

---

## 6. 备份

**导出**会生成 `.zip`（未加密）或 `.platrare`（AES-256 密码加密）文件。您可选择存储位置 — 文件应用、iCloud Drive、Google Drive、AirDrop、本地存储等。**我们从不接收您的备份文件。**

**自动每日备份**仅将文件保存到应用专用设备存储。它**不会**自动上传到任何云服务。您可通过**设置 → 自动每日备份 → 保存到云端**手动将最新备份分享到云位置。

**导入**会将设备上的所有数据替换为所选备份的内容。请仅从您信任并已验证的备份导入。

---

## 7. 儿童

Platrare 不面向 13 岁以下儿童（或您所在司法管辖区适用的最低年龄）。我们不会故意收集儿童的信息。如果您认为儿童不当使用了该应用，请通过应用内的支持详情联系我们。

---

## 8. 数据保留和删除

- 设备上的数据将保留，直到您在应用内将其删除、使用**设置 → 清除数据**、导入替换备份或卸载应用。
- 卸载应用会删除应用本地存储，具体取决于操作系统行为（例如，iCloud 设备备份可能会保留快照，直到被操作系统覆盖）。
- 由于**我们的服务器上没有您数据的副本**，我们这边没有任何内容需要删除。

---

## 9. 您的权利

由于所有数据都存储在您的设备上，您可以通过应用直接行使您的权利：

- **访问和可携带性** — 所有数据在应用内可见。使用**导出备份**获取可携带的、机器可读的副本。
- **更正** — 随时编辑任何账户、交易或类别。
- **删除** — 使用应用内删除功能、**设置 → 清除数据**或卸载应用。

**欧洲经济区和英国用户：** 《通用数据保护条例》（GDPR）和英国 GDPR 可能授予您额外权利，包括向当地数据保护监管机构提出投诉的权利。

**加利福尼亚州居民：** 《加利福尼亚消费者隐私法》（CCPA / CPRA）可能适用。由于我们不按 CCPA 定义出售或共享个人信息，在大多数情况下退出权利不适用。您可以联系我们确认。

---

## 10. 安全

- 所有财务数据存储在设备上其他应用无法访问的**应用沙盒 SQLite 数据库**中。
- 备份文件可使用 **AES-256 加密**和您选择的密码进行保护。
- PIN 仅以**单向加密哈希**形式存储 — 无法恢复或逆向破解。
- 所有网络流量**仅使用 HTTPS**。

---

## 11. 本政策的变更

随着功能的发展，我们可能会更新本政策。顶部的**生效日期**将反映最新修订。在更新后继续使用应用即表示接受修订后的政策。重要变更将在 App Store 和 Google Play 发布说明中注明。

如有隐私相关问题或请求，请使用 App Store 或 Google Play 商店页面上的联系方式，或点击应用内的**设置 → 关于 → 复制支持信息**以获取联系信息。
